ls [filename]


-Lists directory contents(of current directory, if no file name is specified). For files, lists name as well as any requested information associated with the file.

mv [filename] [target directory]


-Moves specified file to target directory.

cp [filename] [target directory]


-Creates a copy of the specified file in the target directory

nano [filename]


-Text editor. Allows users to create/modify code in the specified file. If no file is specified, or the specified file does not exist in the current directory, a new file will becreated(under the specified name).

Login [username]


-Command will require the user to enter the password of the specified account; allows users to login.

rm [filename]


-Deleted target file from inside of current directory.

git init


-Initializes new local git repository

git commit


-Commits any files added/changed(since last commit/push); User must enter a commit message into the text editor that is pulled up.

git pull


-Fetch and merge canged from the remote server to a local working repository.

git push


-Pushes unsynced commits from a local working repository to the remote server.

git add [filename]


-Add the specified file to the local git repository.
